What is the Internet?

The Internet is a global computer network providing a variety of information and communication facilities, consisting of interconnected networks using standardized communication protocols, that allows users to send and recieve data.

History of the Internet

ARPANET: Grandfather to the Internet

The ARPANET, or Advanced Research Projects Agency Network, was a network designed by the US military that connected only a handful of computers at government bases and universities in the Northeastern and Southwestern United States. It allowed those connected to the ARPANET to transmit data to one another in the form of packets, later utilizing TCP/IP. One aspect that led to the success of the ARPANET was fault- tolerancy, meaning if one route for a packet went down, the packet could find an alternate route to it's final destination. This network was later expanded in 1986 to several more universities. This early network of computers paved the way for the internet, which today uses a network sililar to that of the ARPANET, only with millions of nodes instead of just a handful.

DNS: Ease of Access

The DNS, or the Domain Name System, is a naming system that allows computers to translate Domain names (google.com, NSA.gov, etc) into IP addresses, allowing users to access webpages faster and easier than before. The computer does this by sending the domain name to a DNS server, which translates the Domain Name to an IP address, then the server sends the computer to the correct website/server. This system also divided Domain Names into several categories, based on their function and purpose.

  • .com = Company, a for-profit organization
  • .gov = Government Agency
  • .edu = School site, for educational pruposes
  • .net = Organizations involved in networking technologies
  • And Many More

The World Wide Web: The Information Super Highway

The World Wide web was created by an English Scientist, Tim-Berners Lee, in 1989, and was released in 1991. This network allowed users all across the globe to access websites with Universal Reasource Locators. Users could also navigate between websites with the use of hyperlinks. This would replace regional networks like ARPANET, or the french Minitel.
