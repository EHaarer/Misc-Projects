IP Layers Cybersecurity Data and the Cloud

What Is It?

This Layer is the most basic layer of Data Transfer, and deals with bits flowing through physical methods, including wires, radio waves, etc. and deals with the hardware that sends or recieves data on a network.

What Is It?

This Layer deals with the encoding and decoding of packets into bits, and handles errors that occur during the Physical layer. This layer is also split into to sub-layers: The Logic Lik Control and the Media Access Control, controlling error-checking, and dealing with how a user access information on a network, respectively.

What Is It?

This Layer deals with switching and routing of information over the internet across nodes. It also deals with addressing of packets, packet sequencing, and internetworking.

What Is It?

This Layer deals with flow control and the transfer of data between computers, and ensures that there has been a complete data transfer between the two computers.

What Is It?

This Layer deals with establishig and ending connections between two computers. It also manages exchanges and dialogues between the two computers.

What Is It?

This Layer deals with transforming data and information into a form that the application layer can accept. It also formats and Encrypts data that is sent across a network. This allows the data to be free from compatability issues.

What Is It?

This layer identifies what computers are trying to communicate, and if the current network will allow them to communicate. This Layer is then used to send information/recieve information to/from the other computer that is being communicated to. It also checks for user authentication and if the other computer is a reliable source from which to recieve information.
